各サービスでアカウント名が変えれるのか？

髪型変えるのと同じぐらいの頻度でアカウント名たまに変えたくなる。

今回個人事業主になった節目で変えてみたので、2018年時点での各サービスの動向をまとめてみる。

twitter

  • 変更できる
  • 過去の投稿もリダイレクトされる
  • ユーザーページ自体はリダイレクトされない

Github

  • 変更出来る
  • 前のリポジトリはリダイレクトされる。gitのcloneもリダイレクトされる
  • ユーザーページ自体はリダイレクトされない
  • 前のアカウント名はorgとして取得しておくと、URLを乗っ取られたりみたいなことは無い（はず）
  • goのパッケージがどうなるかは試していない
  • [ユーザー名].github.io はリダイレクトされない

qiita

  • 変更出来る
  • githubとほとんど同じ。
  • 記事はリダイレクトはされる
  • ユーザーページはリダイレクトされない

dev.to

  • 変更出来る
  • 記事リダイレクトされる
  • ユーザーページもリダイレクトされる

はてな

  • 変更できない

note

  • 変更できない

medium

  • 変更できない -> 嘘だった。変更できた

npm

  • 変更できない
  • 問い合わせると出来るのかも？（やってない）
  • その場合scopedなパッケージどうなるのかなどは謎

gmail / google account

  • メールそのものなので変更できない（そりゃそうだ）
  • 切り替えは結構だるいけどマルチアカウントログインも出来るのでやろうと思えばそんなに手間ではないかも

macのユーザーアカウント

  • 出来るけどhomeディレクトリが変わって色々中で死ぬのでお勧めしづらい
